# 任务沙盒最小实现（CAP-001）

## 目标

提供 CAP-001 的最小可验收实现：每次任务可创建独立临时目录并在结束时清理，且产出可追溯证据文件。

## 脚本入口

- 创建：`scripts/dev/sandbox-create.sh --smoke --out <dir>`
- 销毁：`scripts/dev/sandbox-destroy.sh --smoke --out <dir>`

## 证据文件

- `sandbox.env`
  - `SANDBOX_ID`
  - `SANDBOX_ROOT`
  - `SANDBOX_MODE`
- `cleanup.log`
  - `sandboxRoot`
  - `removed`
  - `mode`
  - `timestamp`

## 安全约束

- 清理脚本仅允许删除前缀为 `/tmp/helios-sandbox-` 的目录。
- 若 `sandbox.env` 缺失或字段不完整，脚本会记录日志并安全退出，不执行越界删除。
