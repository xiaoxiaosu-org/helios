# 02-架构（索引）

## 本目录解决什么问题

- 引入新基础设施/框架：新增 ADR → 更新 `docs/02-架构/系统总览.md`（并链接 ADR）
- 调整系统边界/依赖方向：新增 ADR → 更新本目录索引与总览
