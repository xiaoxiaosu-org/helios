# PLAN-20260228-02：P0在制项收敛与退场治理计划

## 背景与目标

- 背景：当前执行体系中仅剩旧 P0 总账项（legacy: `TD-001`）长期处于 `in_progress`，且 `closeChecks.move_to_done=false`，形成“可推进但不可收口”的总账项。
