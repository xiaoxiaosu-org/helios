# PLAN-20260228-06：质量评分与演进机制完善

## 背景与目标

- 背景：当前质量评分已接入自动汇总与 CAP 状态同步，但评分口径、阈值解释与演进节奏仍有进一步标准化空间。
- 问题：评分数据源、门禁结果与治理动作之间存在“能看见分数但难以指导演进”的断层，影响持续改进效率。
- 目标：完善质量评分指标体系、校验链路和演进机制，使评分可解释、可纠偏、可持续治理。
- 非目标：不新增平行评分体系，不绕过现有 CI 门禁主链路。

- 阻塞：暂无硬阻塞。
